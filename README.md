# Kütüphane Otomasyon Sistemi

## Proje Hakkında

Bu proje, C programlama dili kullanılarak geliştirilmiş bir kütüphane otomasyon sistemidir. Sistem; kitap, üye ve emanet kitap işlemlerinin yönetimini sağlamaktadır.

## Özellikler

* Kitap ekleme, silme, güncelleme ve arama
* Kitap listeleme
* Üye ekleme, silme, güncelleme ve arama
* Üye listeleme
* Emanet kitap kayıt işlemleri
* Kitap iade ve gecikme ücreti hesaplama
* Bağlı liste (Linked List) yapısı kullanımı
* Dinamik bellek yönetimi (malloc/free)

## Kullanılan Teknolojiler

* C Programlama Dili
* Struct Yapıları
* Linked List
* Pointerlar
* Dinamik Bellek Yönetimi
